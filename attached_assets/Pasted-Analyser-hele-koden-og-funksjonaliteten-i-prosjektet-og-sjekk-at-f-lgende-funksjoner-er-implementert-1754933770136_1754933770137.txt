1. Klienthåndtering (CRM)
- Opprette, redigere og slette klienter.
- To-trinns registrering:
  Trinn 1: Bruker legger inn selskapsnavn og organisasjonsnummer → systemet henter automatisk opplysninger fra Brønnøysund API (org form, adresse, e-post, telefon, postnummer, poststed).
  Trinn 2: Velg AML-status, oppgaver med intervall, regnskapssystem og legg til oppdragsansvarlig (fra ansatte-listen).
- Automatisk utfylling av firmanavn, organisasjonsnummer, adresse, postnummer, poststed, e-post, telefon, næringskode.
- Mulighet for å legge inn notater på klientkortet.
- Mulighet for å legge til ny ansatt (navn, tittel, e-post).

2. Oppgavehåndtering
- Standardoppgaver: Bokføring, MVA, Lønn, Bankavstemming, Kontoavstemming.
- Mulighet for egendefinerte oppgaver (fri tekst).
- Oppgaveintervaller: Ukentlig, Månedlig, Annenhver måned, Årlig, Spesifikk dato.
- Forhåndsdefinerte spesialoppgaver med faste frister:
  - Aksjonærregisteroppgave (desember hvert år)
  - Skattemelding (31. mai hvert år)
  - Årsoppgjør (31. juli hvert år)

3. Timeføring
- Når bruker bytter klient eller logger ut, må timer registreres (antall timer, hvem som jobbet, beskrivelse).
- Samlerapport: timer per ansatt, timer per klient, oppgaver utført.

4. AML/KYC og hvitvasking
- AML-status per klient.
- KYC-sjekklister på klientkort.
- Mulighet for opplasting eller avkryssing av gjennomførte kontroller.

5. Brukerroller
- Admin: kan se total oversikt, fordele oppgaver, se belastning på ansatte.
- Ansatt: kan kun se egne oppgaver og registrere timer.

6. Regnskapssystem-integrasjoner
- Valg av system per klient (Fiken, Tripletex, PowerOffice, Visma, Uni Micro).
- Direkte link til valgt system.

7. AI-assistent
- Aktivert og tilgjengelig for bruk.

8. Integrasjoner
- Brønnøysund API fungerer og returnerer korrekt data.

9. Sjekklister

10. Systemadministrasjon
- Legge til ansatte (navn, tittel, e-post).
